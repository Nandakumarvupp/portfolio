Portfolio

AES-Encrypted Password Manager (Jan 2024 - May 2024)

This project aimed to provide a secure solution for managing passwords in an era where data breaches are increasingly common. I developed a password manager that uses AES-256 encryption to protect user credentials, ensuring that even if the database is compromised, the data remains unreadable without the encryption key. The application integrates SQLite for encrypted storage, with a Tkinter-based GUI for a seamless user experience.

Technologies Used: Python, SQLite, Tkinter, PyCrypto (for AES encryption).
Challenges: One major challenge was securely managing the encryption key to prevent unauthorized access while ensuring usability. I also had to mitigate SQL injection risks by implementing prepared statements in SQLite queries.
Impact: The password manager provides a secure and user-friendly tool for individuals to manage their credentials, reducing the risk of password reuse and enhancing personal security.
Future Improvements: I plan to add multi-factor authentication and cloud synchronization features to enhance accessibility while maintaining security.
