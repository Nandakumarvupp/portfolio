Portfolio

AES-Encrypted Password Manager (Jan 2024 - May 2024)

This project was born out of a need to provide a secure and accessible solution for managing passwords amidst rising cyber threats. I designed and developed a password manager that employs AES-256 encryption to safeguard user credentials, ensuring data remains secure even in the event of a breach. SQLite serves as the encrypted storage backend, and I built a user-friendly GUI using Tkinter for seamless interaction.

Technologies Used: Python, SQLite, Tkinter, PyCrypto (for AES encryption).
Challenges: Securely managing the encryption key was a significant challenge, as was preventing SQL injection vulnerabilities. I implemented secure key storage practices and used prepared statements to mitigate risks.
Impact: The tool empowers users to manage their passwords securely, reducing the risk of password reuse and enhancing personal cybersecurity. It has been tested with a small user group, receiving positive feedback on its usability and security features.
Future Improvements: I plan to integrate multi-factor authentication, cloud synchronization with end-to-end encryption, and a password strength analyzer to further enhance the tool's capabilities.

Network Vulnerability Scanner (Aug 2024 - Dec 2024)

With the increasing complexity of network environments, I developed a vulnerability scanner to help administrators identify and mitigate security risks. The tool leverages Nmap for port and service scanning and Wireshark for network traffic analysis, with added features for basic intrusion detection to alert users of potential threats like unauthorized port scans.

Technologies Used: Python, Nmap, Wireshark.
Challenges: Parsing Nmap's verbose output into actionable insights was complex, and handling large volumes of network traffic data required optimization to prevent performance issues.
Impact: The scanner has proven effective in identifying vulnerabilities in small-scale networks, enabling administrators to take proactive measures. It has been used in a university lab setting to secure test networks.
Future Improvements: Future iterations will include machine learning models for improved intrusion detection, automated remediation suggestions, and a web-based dashboard for easier access to scan results.
