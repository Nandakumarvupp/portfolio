Portfolio

AES-Encrypted Password Manager (Jan 2024 - May 2024)

I developed a secure password manager to address the growing need for data protection in personal applications. The project utilized AES-256 encryption to ensure the confidentiality of user credentials, with SQLite as the backend for encrypted storage. I implemented a user-friendly GUI using Tkinter, allowing seamless interaction for users to store, retrieve, and manage their passwords securely.

Technologies Used: Python, SQLite, Tkinter, PyCrypto (for AES encryption).
Challenges: Ensuring secure key management for AES encryption and preventing SQL injection in SQLite queries.
Outcome: Successfully created a functional password manager with robust encryption, capable of securely storing user credentials and providing a simple interface for password management.

Network Vulnerability Scanner (Aug 2024 - Dec 2024)

I built a network vulnerability scanner to identify potential security risks in network environments. The tool uses Python and Nmap to scan for open ports and services, while Wireshark is employed to analyze network traffic and detect vulnerabilities. The scanner also includes basic intrusion detection features to alert users of potential threats.

Technologies Used: Python, Nmap, Wireshark.
Challenges: Parsing and interpreting Nmap scan results accurately, and handling large volumes of network traffic data in Wireshark.
Outcome: Delivered a reliable tool that identifies network vulnerabilities, provides actionable insights, and supports basic intrusion detection, enhancing network security for small-scale environments.

Securing the Cloud: Best Practices for Azure and Microsoft 365

Cloud security is more critical than ever with the rise of remote work and digital transformation. In this blog, I explore key strategies to secure Azure and Microsoft 365 environments. This project highlights practical applications of cryptography in everyday tools.
