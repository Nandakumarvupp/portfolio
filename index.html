Securing the Cloud: Best Practices for Azure and Microsoft 365

Cloud security is more critical than ever with the rise of remote work and digital transformation. In this blog, I explore key strategies to secure Azure and Microsoft 365 environments. Topics include implementing robust Public Key Infrastructure (PKI), using TLS/SSL for secure communication, and leveraging tools like Splunk for monitoring. I also discuss certificate lifecycle management and how to use Active Directory Certificate Services (ADCS) to enhance security.

Understanding AES Encryption: Building a Password Manager

AES-256 encryption is a cornerstone of modern cybersecurity. In this blog, I walk through my project of building a password manager using Python, SQLite, and Tkinter. I explain how AES encryption ensures data confidentiality, the role of secure authentication, and the importance of encrypted credential storage. This project highlights practical applications of cryptography in everyday tools.
